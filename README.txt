
Sempre Biryani - React (Vite) project (v3 full)
----------------------------------------------

How to run:
  1. cd sempre-biryani-v3-full
  2. npm install
  3. npm run dev

Notes:
  - Default language is Italian. Use the IT / EN buttons in the navbar to switch languages.
  - Replace any Formspree endpoints if needed. Contact is already set to your existing endpoint.
  - To change hero images, edit src/components/Hero.jsx images array.
